Get started with Camunda Platform 7

There are different ways to get started with Camunda Platform 7. Choose from the following guides:
Quick Start (Java / JS)
Learn how to model a BPMN 2.0 process using the Camunda Modeler, adding a Java or JavaScript Worker. Deploy it to the Camunda Platform.
Spring Boot
Get started with using Camunda Platform in Spring Boot application.
DMN
Learn how to create a DMN 1.3 decision table using the Camunda Modeler. Package it as a web application and deploy it to an Apache Tomcat Server.
